Allen Disk 已經停止維護

很遺憾的通知您，Allen Disk 已經停止維護，請不要再使用這套程式，我們建議您改用功能更完整且更安全的 NextCloud。目前 Allen Disk 的程式碼有許多 Bug 與漏洞，強烈建議不要再使用，若您堅持要使用 Allen Disk，請自行負責一切風險。

其實 Allen Disk 只是我國中時想練習 PHP 時寫的小 Project，以現在回頭看，這個程式的 Code 髒亂，很多不安全的寫法，效能低落，太多 Workaround，不論以什麼角度去看，這都不會是個好程式。我從未想過這個程式會受到大家關注，甚至還拿過 CVE（笑），由於我相信有遠比 Allen Disk 更好用的雲端硬碟軟體，就讓這個國中生隨便寫的東西安息吧，感謝各位的支持！

— 開發者 Allen Chou
